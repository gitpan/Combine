\paragraph{NAME\label{NAME}\index{NAME}}


XWI.pm - class for internal representation of a document record

\paragraph{SYNOPSIS\label{SYNOPSIS}\index{SYNOPSIS}}
\begin{verbatim}
 use Combine::XWI;
 $xwi = new Combine::XWI;
\end{verbatim}
\begin{verbatim}
 #single value record variables
 $xwi->server($server);
\end{verbatim}
\begin{verbatim}
 my $server = $xwi->server();
\end{verbatim}
\begin{verbatim}
 #original content
 $xwi->content(\$html);
\end{verbatim}
\begin{verbatim}
 my $text = ${$xwi->content()};
\end{verbatim}
\begin{verbatim}
 #multiple value record variables
 $xwi->meta_add($name1,$value1);
 $xwi->meta_add($name2,$value2);
\end{verbatim}
\begin{verbatim}
 $xwi->meta_rewind;
 my ($name,$content);
 while (1) {
  ($name,$content) = $xwi->meta_get;
  last unless $name;
 }
\end{verbatim}
\paragraph{DESCRIPTION\label{DESCRIPTION}\index{DESCRIPTION}}


Provides methods for storing and retrieving structured records
representing crawled documents.

\paragraph{METHODS\label{METHODS}\index{METHODS}}
\subparagraph*{new()\label{new_}\index{new()}}
\subparagraph*{XXX(\$val)\label{XXX_val_}\index{XXX(\$val)}}


Saves \$val using AUTOLOAD. Can later be retrieved, eg

\begin{verbatim}
    $xwi->MyVar('My value');
    $t = $xwi->MyVar;
\end{verbatim}


will set \$t to 'My value'

\subparagraph*{*\_reset()\label{_reset_}\index{*\ reset()}}


Forget all values.

\subparagraph*{*\_rewind()\label{_rewind_}\index{*\ rewind()}}


*\_get will start with the first value.

\subparagraph*{*\_add\label{_add}\index{*\ add}}


stores values into the datastructure

\subparagraph*{*\_get\label{_get}\index{*\ get}}


retrieves values from the datastructure

\subparagraph*{meta\_reset() / meta\_rewind() / meta\_add() / meta\_get()\label{meta_reset_meta_rewind_meta_add_meta_get_}\index{meta\ reset() / meta\ rewind() / meta\ add() / meta\ get()}}


Stores the content of Meta-tags



Takes/Returns 2 parameters: Name, Content

\begin{verbatim}
 $xwi->meta_add($name1,$value1);
 $xwi->meta_add($name2,$value2);
\end{verbatim}
\begin{verbatim}
 $xwi->meta_rewind;
 my ($name,$content);
 while (1) {
  ($name,$content) = $xwi->meta_get;
  last unless $name;
 }
\end{verbatim}
\subparagraph*{xmeta\_reset() / xmeta\_rewind() / xmeta\_add() / xmeta\_get()\label{xmeta_reset_xmeta_rewind_xmeta_add_xmeta_get_}\index{xmeta\ reset() / xmeta\ rewind() / xmeta\ add() / xmeta\ get()}}


Extended information from Meta-tags. Not used.

\subparagraph*{url\_remove() / url\_reset() / url\_rewind() / url\_add() / url\_get()\label{url_remove_url_reset_url_rewind_url_add_url_get_}\index{url\ remove() / url\ reset() / url\ rewind() / url\ add() / url\ get()}}


Stores all URLs (ie if multiple URLs for the same page) for this record



Takes/Returns 1 parameter: URL

\subparagraph*{heading\_reset() / heading\_rewind() / heading\_add() / heading\_get()\label{heading_reset_heading_rewind_heading_add_heading_get_}\index{heading\ reset() / heading\ rewind() / heading\ add() / heading\ get()}}


Stores headings from HTML documents



Takes/Returns 1 parameter: Heading text

\subparagraph*{link\_reset() / link\_rewind() / link\_add() / link\_get()\label{link_reset_link_rewind_link_add_link_get_}\index{link\ reset() / link\ rewind() / link\ add() / link\ get()}}


Stores links from documents



Takes/Returns 5 parameters: URL, netlocid, urlid, Anchor text, Link type

\subparagraph*{robot\_reset() / robot\_rewind() / robot\_add() / robot\_get()\label{robot_reset_robot_rewind_robot_add_robot_get_}\index{robot\ reset() / robot\ rewind() / robot\ add() / robot\ get()}}


Stores calculated information, like genre, language, etc



Takes/Returns 2 parameters Name, Value. Both are strings with max length Name: 15, Value: 20

\subparagraph*{topic\_reset() / topic\_rewind() / topic\_add() / topic\_get()\label{topic_reset_topic_rewind_topic_add_topic_get_}\index{topic\ reset() / topic\ rewind() / topic\ add() / topic\ get()}}


Stores result of topic classification.



Takes/Returns 5 parameters: Class, Absolute score, Normalized score, Terms, Algorithm id



Class, Terms, and Algorithm id are strings with max
lengths Class: 50, and Algorithm id: 25



Absolute score, and Normalized score are integers



Normalized score and Terms are optional and may be replaced with 0, and '' respectively
